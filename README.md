## **✨ 核心特性 (Features)**

### **1\. 🖥️ 沉浸式海报工坊 2.0 (Immersive Poster Studio)**

全新重构的独立创作空间，带来更专业、更自由的海报制作体验。

*   **🎨 风格化创作**:
    *   **三种独家模版**：
        *   **经典沉浸 (Classic)**：全屏通透大图，强调文字张力，适合风景与意境图。
        *   **文艺拍立得 (Polaroid)**：优雅留白与复古胶片感，记录生活瞬间。
        *   **电影故事 (Cinema)**：宽画幅黑边 + 电影字幕特效，赋予画面叙事感。
    *   **✨ 滤镜支持**：内置多款精选滤镜，一键调出完美色调。
*   **📐 自由构图**:
    *   **多比例支持**：支持 **9:16 (手机壁纸)**、**1:1 (Instagram)**、**3:4**、**4:3** 等多种比例，完美适配各类社交平台。
    *   **图片编辑**：支持**本地上传**、自由拖拽移动、滚轮缩放与滤镜叠加。
*   **🎛️ 全能控制**:
    *   **模块化面板**：排版（字号/行距）、图片、导出设置分区管理，操作更直观。
    *   **繁简切换 (New)**：支持一键切换歌词的**简体中文/繁体中文**显示，完美保留原词韵味。
    *   **智能换行控制**：
        *   **默认模式**：歌词中的**空格**与**回车**均会触发换行，适合短句排版。
        *   **紧凑模式**：开启后，空格不再触发强制换行，仅在手动**回车**或文本超出画布宽度时换行。如："**你 何以始终不说话 尽管讲出不快吧**"，默认模式下会自动换行为三行歌词，若勾选紧凑模式，可自行在"**尽管讲出不快吧**"前换行，这样可控制在两行
    *   **多主题支持 (New)**：基于陈奕迅经典专辑配色的5款应用主题
        *   **CHIN UP!**：清爽蓝调，充满活力与正能量
        *   **DUO**：经典香槟金，温暖优雅
        *   **L.O.V.E.**：都市牛仔蓝，现代简约
        *   **FEAR&DREAMS**：现代画廊灰，深沉玫瑰红
        *   **THE KEY**：复古米色，深邃暗红
    *   **水印控制**：支持自定义开启/关闭 "Eason Moment" 品牌水印。
*   **📤 高清导出**:
    *   **高清渲染**：优化 Canvas 引擎，导出画质更细腻。
    *   **仪式感**：保留了经典的旋转/脉动加载特效，让等待也成为享受。

## **🚀 安装与使用 (Installation)**

### **开发环境运行**

```bash
# 1. 克隆项目
git clone https://github.com/Gaofan1874/eason-moment.git

# 2. 安装依赖 (推荐使用 npmmirror)
npm config set registry https://registry.npmmirror.com
export ELECTRON_MIRROR="https://npmmirror.com/mirrors/electron/"
npm install

# 3. 启动应用
npm run dev
```

### **打包应用**

```bash
# 构建 macOS 应用 (.dmg)
npm run build

# 构建 Windows 应用 (.exe)
# 推荐使用 GitHub Actions 进行构建，本地构建需配置 Wine 环境
npm run build -- --win --x64
```

打包完成后，你可以在 `dist/` 目录下找到安装包 (`.dmg` 或 `.exe`)。

### **⚠️ 常见问题 (FAQ)**

#### **macOS 提示“应用已损坏，无法打开”？**
由于应用未经过 Apple 开发者签名，从 GitHub 下载后会被系统 Gatekeeper 拦截。请在终端执行以下命令即可正常打开：

```bash
sudo xattr -r -d com.apple.quarantine /Applications/eason-moment.app
```
