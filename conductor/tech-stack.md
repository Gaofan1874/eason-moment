# Tech Stack - Eason Moment

## 1. Core Frameworks
- **Electron (28.3.3)**: 桌面端核心容器，负责主进程管理与跨平台窗口分发。
- **React (18.2.0)**: 渲染进程前端框架，构建交互式 UI。

## 2. Languages & Tooling
- **TypeScript (5.0.2)**: 全栈类型安全支持。
- **Vite (5.0.0)**: 高性能前端构建与 HMR 开发环境。
- **Electron Builder (24.9.1)**: 应用打包与分发工具。

## 3. UI & UX Libraries
- **CSS Modules**: 局部作用域样式管理，防止样式污染。
- **Framer Motion (10.16.4)**: 负责复杂的 UI 动效与过渡动画。
- **Lucide Icons (0.292.0)**: 轻量化矢量图标库。
- **Native Canvas API**: 海报工坊的核心引擎，负责高清图片渲染与导出。

## 4. Testing & Quality
- **Vitest (1.6.1)**: 高性能单元测试运行器。
- **React Testing Library**: 组件化功能测试。
- **Prettier**: 代码格式化规范。
