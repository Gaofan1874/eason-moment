# Initial Concept
Eason Moment 是一款专为陈奕迅粉丝打造的沉浸式桌面应用，旨在以优雅、独立的方式将歌词融入用户的桌面生活。

# Product Guide - Eason Moment

## 2. Target Users
- **陈奕迅的核心粉丝**：希望收藏、珍藏经典歌词，并制作精美海报分享至社交平台的资深乐迷。
- **追求氛围感的音乐爱好者**：希望在工作或闲暇时，通过桌面歌词悬浮窗或菜单栏，静默地与音乐产生情感共鸣的用户。
- **内容创作者**：需要一款高效、专业且具有设计感的工具，快速生成文艺范儿歌词图片的创作者。

## 3. Core Goals & Features
- **沉浸式海报工坊 2.0 (Immersive Poster Studio)**：
    - 提供多种艺术风格模板（经典沉浸、文艺拍立得、电影故事）。
    - 支持多比例调整（9:16, 1:1, 3:4, 4:3）及高清导出。
    - 深度定制功能：内置滤镜、繁简中文一键切换、智能换行控制、品牌水印管理。
- **智能托盘与桌面歌词 (Smart Desktop Lyrics)**：
    - **双平台适配**：macOS 菜单栏歌词与 Windows 桌面悬浮窗。
    - **情感联动**：提供六大心情模式（随机漫步、爱与浪漫、深夜抑郁、治愈哲理、浮夸热血、岁月金曲）。
    - **个性化交互**：支持自定义轮播节奏与颜色样式。
- **跨平台一致性体验**：
    - 针对不同操作系统深度优化（如 macOS 的毛玻璃特效与 Windows 的置顶悬浮窗），确保原生般的丝滑体验。

## 4. Design Principles
- **极简主义与沉浸感**：去除繁杂干扰，强调文字的视觉张力与图片意境的融合。
- **情感化与仪式感**：通过心情分类与精选专辑配色主题（如 CHIN UP!, DUO, THE KEY），为用户营造独特的心理氛围，连加载特效也充满仪式感。
- **原生质感与优雅交互**：追求与操作系统风格的一致性，通过平滑的过渡动画与高品质的 UI 控件，提升应用的整体品质感。
